# LINE NUMBER MATCH

同じ数字または合計10になるよう数字を直線で繋げて消すゲーム。

5ターン経過するか繋げれる数字が無くなったらラインが追加されます。

一番上にラインがある状態でラインが追加されるとゲームオーバー。
